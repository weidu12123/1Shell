# 板块四：文件管理

---

## 1. 打开文件浏览器

文件浏览器位于左侧边栏的下半部分。点击「文件浏览」标题即可展开。

支持两种模式：
- **本地模式**：浏览 1Shell 服务所在机器的文件系统
- **远程模式（SFTP）**：浏览当前已连接的远程主机的文件系统

切换到不同主机的终端时，文件浏览器自动切换到对应主机的文件系统。

---

## 2. 目录浏览

![文件浏览器](images/04-file-browser.png)

### 2.1 界面元素

- **路径栏**：显示当前目录路径（如 `/ > root`），点击路径中的任意层级可快速跳转
- **上传按钮**：上传文件到当前目录
- **返回上级**（`↑ ..`）：返回上一层目录
- **文件/文件夹列表**：黄色文件夹图标表示目录，文件图标表示文件

### 2.2 操作

- **点击文件夹**：进入该目录
- **点击文件**：打开文件预览
- 显示隐藏文件（以 `.` 开头的文件和文件夹）

---

## 3. 文件预览与编辑

点击任意文件后，打开文件预览界面。

![文件预览](images/04-file-preview.png)

### 3.1 预览界面

- **文件名**：顶部显示文件名称（如 `1paneld.procd`）
- **完整路径**：文件名下方显示文件的完整路径
- **文件内容**：代码高亮显示文件内容
- **文件大小**：底部显示文件大小（如 `562 B`）

### 3.2 操作按钮

预览界面右上角提供以下操作：

| 按钮 | 功能 |
|------|------|
| **路径** | 复制文件的完整路径到剪贴板 |
| **下载** | 将文件下载到本地电脑 |
| **编辑** | 进入在线编辑模式，可直接修改文件内容并保存 |
| **×** | 关闭预览 |

---

## 4. 文件上传

点击文件浏览器顶部的 **「上传」** 按钮，选择本地文件上传到当前浏览的目录。

- 支持上传任意类型的文件
- 文件上传到当前目录路径
- 本地模式上传到 1Shell 所在机器
- 远程模式通过 SFTP 上传到远程主机

---

## 5. 在线编辑

在文件预览界面点击 **「编辑」** 按钮进入编辑模式：

1. 文件内容变为可编辑状态
2. 直接修改文件内容
3. 点击「保存」将修改写入文件

适用于快速修改配置文件、脚本等文本文件，无需通过终端使用 `vim` 或 `nano`。

---

## 6. 本地与远程对比

| 特性 | 本地模式 | 远程模式（SFTP） |
|------|---------|----------------|
| **文件系统** | 1Shell 服务器本机 | 远程 SSH 主机 |
| **传输协议** | 直接文件系统操作 | SFTP over SSH |
| **上传** | 支持 | 支持 |
| **下载** | 支持 | 支持 |
| **在线编辑** | 支持 | 支持 |
| **切换方式** | 选择本机终端标签 | 选择远程主机终端标签 |

文件浏览器与终端共享同一个 SSH 连接，不需要额外配置 SFTP 端口或权限。
