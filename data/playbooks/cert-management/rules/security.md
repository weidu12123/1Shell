# 安全规则

## 红旗警告（遇到以下情况立即停止，报告给用户）
- 操作涉及删除 `/etc/ssh/` 下任何文件
- 操作涉及修改 sshd_config
- 操作涉及修改防火墙规则（除非用户明确要求开放 80/443 端口）
- 私钥文件权限不是 600

## 私钥保护
- 私钥文件权限必须设为 `600`
- 私钥所有者必须是 `root:root`
- 绝对不要在输出中打印私钥内容
- 不要将私钥传输到其他主机

## API Token 保护
- Cloudflare API Token 只通过环境变量传入，不写入命令行参数
- 不要在输出中回显 Token 值
- 配置文件中如果包含 Token，权限设为 `600`

## 已有证书保护
- 不得删除主机上已有的其他域名证书
- 不得覆盖其他域名的证书文件
- 操作前检查目标目录是否已有其他域名的证书
